# ①課題名
じゃんけんの向こう側へ…

## ②課題内容（どんな作品か）
じゃんけんに負けが込む（3回）と格闘モードボタンが出現し、ルール無用のバトル展開になります。

## ③アプリのデプロイURL
デプロイしている場合はURLを記入（任意）

## ⑤工夫した点・こだわった点
じゃんけんで納得できなかったら、もはや暴力しかなくなるという民度低めの展開にしてみたかったのですが、表面上は形にできたと思います。
体力ゲージもそれっぽく入れられました。

## ⑥難しかった点・次回トライしたいこと（又は機能）
体力ゲージに関数を入れて計算したかたのですが、全然うまくいかず。
結局、if文で無理やり10回判定させて、敵の画像と体力ゲージは決め打ちで表示を切り替える力技で力尽きました。
ボタンは何を押しても10回で終了です。
終了後、リロード以外初期画面に戻る方法がないので、リセットするボタンを実装したいです。

## ⑦フリー項目（感想、シェアしたいこと等なんでも）
画像の置き換えがうまくいかず、何回も何回も記述をくりかえしていたら、突然反映されるようになりました。おそらく記述ミスだったと思われます。焦りますが、落ち着いて見直すことが大事だと思いました。
